# Privacy Policy — NNG Cinema Studio

Effective date: 2026-02-08

NNG Cinema Studio is an educational simulation application that demonstrates how a cinema booking system can be architected using Flutter.

This application is created for learning and demonstration purposes only.

## Data Collection

NNG Cinema Studio does NOT collect, store, or share any personal user data.

- No user registration is required
- No personal information is collected
- No payment information is requested
- No location data is collected
- No background tracking is performed

## Use of Third-Party Services

This app may demonstrate integration patterns with public APIs such as:

- TMDB API (public movie data)
- OpenWeatherMap API (demo logic)
- Firebase services (demonstration only)

These services are used strictly for educational demonstration and do not collect personal user information through this app.

## Cookies and Tracking

NNG Cinema Studio does not use cookies or tracking technologies.

## Children’s Privacy

This app is not directed to children under the age of 13. No personal data is collected from any users.

## Security

As no personal data is collected, there is no storage of sensitive user information.

## Changes to This Policy

This privacy policy may be updated for clarity as the project evolves.
